- описывает класс с некоторым внутренним классом согласно варианту (таблица 1)
- демонстрирует использование обоих классов

Данные для объектов классов задаются вводом с клавиатуры
Информация об объектах выводится в консоль в табличном виде
Для демонстрации использования классов необходимо организовать поиск каких-либо данных

Внешний класс: Справочная_городского_транспорта: содержит записи о проезде на транспорте
Внутренний класс: Запись: содержит описание линии маршрутов, времени функционирования, стоимости
