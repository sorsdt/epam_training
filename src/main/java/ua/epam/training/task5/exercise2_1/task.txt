Напишите консольное приложение Java, которое рекурсивно обрабатывает указанную директорию согласно варианта, запуская
для каждой поддиректории отдельный поток, с использованием дополнительных возможностей по синхронизации и блокировки
доступа к ресурсам

Обеспечить обработку произвольной директории в файловой системе компьютера, для чего ее местоположение и название ввести
    набором с клавиатуры
При необходимости название новых директории или файла для записи результата поиска тоже ввести набором на клавиатуре
Для обеспечения устойчивости работы применения предусмотреть проверку корректности ввода данных и предложить решение в
    случае неверного их введения для продолжения работы приложения
Все действия должны фиксироваться выводом соответствующих сообщений

Найти в заданной директории текстовые файлы, содержащие действительные значения, и подсчитать их количество в каждом
найденном файле. В некоторый файл записать названия таких файлов и количество чисел. Вывести в консоль содержимое
созданного файла
